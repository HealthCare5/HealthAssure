COVID-19 Detection

COVID-19 is a highly contagious disease, and has become a global epidemic that needs to be addressed as soon as possible. Due to its rapid speed of spreading comes the need for a system which can be used to detect the virus.
With the increase in use of technology, lots of data about COVID-19 is readily available at our fingertips, which can be used to obtain important information about the virus. In this project, we compared the accuracies of different machine learning algorithms in predicting COVID-19 and used the most accurate one in the final model testing.

Wanna check you may have covid-19 or not?

This is a machine learning model which will predict that you have covid-19 or not.It is based on Random Forest Classifier Algorithm.
